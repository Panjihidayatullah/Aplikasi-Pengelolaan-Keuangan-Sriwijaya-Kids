# Aplikasi-Pengelolaan-Keuangan-Sriwijaya-Kids
Aplikasi Pengelolaan Keuangan berbasis website yang dibangun untuk Homeschooling Sriwijaya Kids. Sistem ini dibangun menggunakan Laravel dan PostgreSQL untuk membantu pencatatan pemasukan, pengeluaran, serta laporan keuangan secara terstruktur, efisien, dan mudah diakses.
